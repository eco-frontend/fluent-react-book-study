# fluent-react-book-study

## 북스터디 규칙 (정확한 룰은 RULES.md 참고)
1. 다음주에 이전주의 액션아이템을 실천했다면 경험을 공유해보기, 이야기 나눠보기
    - 실습 적용 + 
2. 문장 혹은 이야기하고 문단까지 읽어보기, 한 페이지씩 읽어보기 (챕터는 유동적으로) 
    - 사람이 모자르다면 2인 기준 최대 3페이지까지 읽어보기
3. 각 챕터마다 마지막에 회고 하고 액션아이템 찾아보기
